Spring-Service
--------------

A program that demonstrates using Spring features from inside a regular Java application.

Features
~~~~~~~~

* A plain Java service bean
* MongoDB beans
  - MongoTemplate
  - MongoRepository
* Redis beans
  - CrudRepository
  - Jedis
  - RedisTemplate
