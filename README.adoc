Spring-Service
--------------

A program that demonstrates using Spring features from inside a regular Java application.

Features
~~~~~~~~

* Plain Java service bean
  - HelloWorldService
  - StringService
* MongoDB beans
  - MongoTemplate
  - MongoRepository
* Redis beans
  - CrudRepository
  - Jedis
  - Redisson
  - RedisTemplate
